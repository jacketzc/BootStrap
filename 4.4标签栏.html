使用js手动激活

$('#myTabs a').click(function (e) {
  e.preventDefault()
  $(this).tab('show')
})

激活标签的几种方式

$('#myTabs a[href="#profile"]').tab('show') //使用名称选择
$('#myTabs a:first').tab('show') // 选择第一个标签
$('#myTabs a:last').tab('show') // 选择最后一个标签
$('#myTabs li:eq(2) a').tab('show') // 选择第三个标签 (标签从0开始计数)

通过data属性调用

为每个链接添加控制属性：

data-toggle="tab"

为目标控制组添加控制样式

class="tab-content"

为每个目标控件添加控制样式：

class="tab-pane"
